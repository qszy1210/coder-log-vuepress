# 数组删除多个元素

1. 逆向排查删除

采用 splice方法, 正向删除是不可能的了, 这辈子都不可能的了, 因为删除后会引起数组的长度和index的变化

```
const arr = [1,2,3,4,4,5];
const target = 4;
for(let index = arr.length -1; index > 0; index--) {
    if (arr[index] === target) {
        arr.splice(index, 1);
    }
}
```
2. filter方法进行查询

```
let arr = [1,2,3,4,4,5];
arr = arr.filter(item=>item!==4);
```
